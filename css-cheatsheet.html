CSS Properties

Here are 15 essential CSS properties and their functions:

  1. color: Sets the color of text. Example: color: red;
  2. background-color: Sets the background color of an element. Example: background-color: #f0f0f0;
  3. font-size: Specifies the size of the font. Example: font-size: 16px;
  4. font-family: Sets the typeface for the text. Example: font-family: Arial, sans-serif;
  5. padding: Defines space inside an element, between its content and border. Example: padding: 20px;
  6. margin: Sets space outside an element, creating distance between it and neighboring elements. Example: margin: 15px;
  7. border: Defines the border around an element. Example: border: 1px solid black;
  8. width: Sets the width of an element. Example: width: 100px;
  9. height: Sets the height of an element. Example: height: 50px;
  10. display: Specifies how an element should be displayed (block, inline, etc.). Example: display: block;
  11. position: Specifies the positioning method of an element (static, relative, absolute, fixed). Example: position: absolute;
  12. top, right, bottom, left: Used with positioned elements to define their position within a containing element. Example: top: 10px;
  13. flex: A shorthand for setting flex properties on a container in a flexbox layout. Example: flex: 1;
  14. grid: Defines a grid-based layout for a container. Example: display: grid;
  15. text-align: Aligns text within an element (left, right, center, justify). Example: text-align: center;
